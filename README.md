#### 安装

1. 拉取项目

   ``` bash
   git clone https://github.com/weijin-top/online-recruitment-system-backend.git
   ```

2. 导入IDEA，配置好maven、java版本（需要java17版本）

3. 配置application.yml，配置好mysql、redis和阿里云OSS。

4. 直接点运行就可以使用了
